# Análise de Eficiência em Compras

# Visão Geral

Este projeto apresenta um dashboard interativo desenvolvido no Power BI com o objetivo de analisar a eficiência do processo de compras de uma organização. A análise contempla indicadores estratégicos que permitem avaliar desempenho, identificar oportunidades de redução de custos e apoiar a tomada de decisão baseada em dados.

# Objetivos

- Monitorar o desempenho do setor de compras
- Identificar variações de preços entre fornecedores
- Avaliar eficiência de compradores
- Detectar oportunidades de otimização de custos
- Apoiar decisões estratégicas com base em dados

# Principais Indicadores (KPIs)

- Valor total de compras
- Quantidade de pedidos realizados
- Ticket médio por compra
- Economia gerada (%)
- Performance por fornecedor
- Performance por comprador

# Estrutura dos Dados

O modelo de dados foi estruturado seguindo boas práticas de modelagem dimensional:

_Tabelas Fato:_

- fCompras

_Tabelas Dimensão:_

- dFornecedor
- dComprador
- dProduto
- dData

# Ferramentas Utilizadas

- Power BI
- Excel (para geração e tratamento dos dados)
- Modelagem dimensional (Star Schema)
- DAX para criação de métricas e indicadores

# Análises Realizadas

- Comparativo de preços entre fornecedores
- Análise de eficiência por comprador
- Evolução temporal das compras
- Identificação de padrões de consumo
- Avaliação de economia gerada

# Insights Obtidos

- Identificação de fornecedores com maior custo médio
- Compradores com melhor desempenho em negociação
- Períodos com maior volume de compras
- Oportunidades de padronização de fornecedores

# Possíveis Melhorias

- Integração com dados em tempo real
- Inclusão de previsões (forecasting)
- Implementação de indicadores de SLA
- Análise de lead time de compras
